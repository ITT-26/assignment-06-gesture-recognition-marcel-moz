# Task 3 Gesture Detection Game
- The files for task 3 are in [`./03-gesture-game/`](./03-gesture-game/).
- The rock, paper ,scissors game needs to be started via running the [`gesture_application.py`](./03-gesture-game/gesture_application.py) file e. g. `python .\gesture_application.py`.
- When the game is running, you can draw the gestures `delete`, `rectangle` and `circle`, in the gray rectangle to choose scissors, paper and rock (same order).
- The used model is saved as [`gesture_recognizer.keras`](./03-gesture-game/gesture_recognizer.keras).
- The model was trained in the notebook [`recognizer_training.ipynb`](./03-gesture-game/recognizer_training.ipynb).
  The notebook needs the XML-gesture logs from the $1-recognizer. They need to be in the same directory and the name has to be `xml_logs`.
